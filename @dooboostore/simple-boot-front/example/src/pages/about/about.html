ℹ️ About Simple Boot Front

Simple Boot Front is a powerful web framework that combines the best features of Simple Boot and Dom Render, providing a complete solution for building modern single-page applications.

It offers dependency injection, component-based architecture, routing, and reactive data binding out of the box.

🛠️ Built With:

  • 📦 @dooboostore/simple-boot - DI Container & AOP
  • 🎨 @dooboostore/dom-render - Reactive Rendering
  • 🔧 @dooboostore/core - Core Utilities
  • ⚡ Webpack 5 - Module Bundler
  • 📘 TypeScript - Type Safety
