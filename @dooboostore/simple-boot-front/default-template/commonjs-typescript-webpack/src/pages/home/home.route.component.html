Project Structure

  • pages/ - Router and route pages
  • components/ - Reusable components (*.component.ts)
  • services/ - Business logic and data services
  • utils/ - Utility functions and helpers

Naming Convention

  • *.router.component.ts - Router with Component (extends ComponentRouterBase)
  • *.route.component.ts - Route with Component (extends ComponentBase)
  • *.component.ts - Pure Component (extends ComponentBase)
